# Meus Estudos e Pequenos Projetos em Python

## Sobre este Repositório
Para manter o meu perfil organizado e focado nas melhores práticas de desenvolvimento, decidi centralizar todos os meus exercícios de lógica, treinos de sintaxe e pequenos projetos resultantes de exercícios em livros, bootcamps e cursos neste repositório único de estudos.

Em vez de criar dezenas de repositórios pequenos e isolados, cada pasta aqui dentro representa um módulo, uma aula ou um desafio.

## Tecnologias e Ferramentas Utilizadas
*   **Linguagem:** Python 3 (focado em lógica limpa e legibilidade)
*   **Editor de Código:** Visual Studio Code (VS Code)
*   **Controle de Versão:** Git e GitHub para versionamento de código
